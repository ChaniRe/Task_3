# מדריך התקנה - TaskFlow AI

ברוכים הבאים ל-TaskFlow! עקבו אחר השלבים הפשוטים הבאים כדי להגדיר את סביבת העבודה שלכם ולהתחיל לנהל משימות בצורה חכמה.

## דרישות מערכת
לפני שמתחילים, וודאו שהכלים הבאים מותקנים אצלכם:
* **Node.js**: גרסה 18.0.0 ומעלה.
* **מנהל חבילות**: npm או yarn.

---

## שלבי ההתקנה

### 1. התקנת חבילת ה-CLI
פתחו את הטרמינל והריצו את הפקודה הבאה כדי להתקין את הכלי באופן גלובלי:

```bash
npm install -g taskflow-ai-cli
